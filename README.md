# Moon-Center-Seccion-de-Informatica-v2.0
Moon Center – Sección de Informática v2.0. Proyecto web educativo orientado a la enseñanza de los fundamentos de la informática, incluyendo hardware, software y programación. Desarrollado en HTML5 y CSS3, optimizado para dispositivos móviles y de escritorio. Creado y mantenido por Marcos.
